# a-geocat
https://projetoalfobre.github.io/a-geocat/

A-GeoCat (Alfobre GeoData Catalog) is an open geospatial data catalog and it’s a project from Alfobre.com

Our datasets are licensed by Alfobre.com under Attribution 4.0 International.

Alfobre (pt for seedbed) is a collective devoted to the creation and development of projects, strategies and products in the Heritage field.
